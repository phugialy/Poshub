## How to Use

1. **Add Tracking**
   - Enter tracking number (auto-detects carrier)
   - Optional: Select carrier manually
   - Click "Track Package"

2. **Search & Filter**
   - Use search bar to find specific packages
   - Use filters to narrow down results
   - Results update in real-time

3. **Manage Trackings**
   - **View**: Click "View" to open carrier tracking page
   - **Edit**: Click edit icon to update description
   - **Delete**: Click delete icon to remove tracking

4. **View Stats**
   - Dashboard shows overview stats
   - Updates automatically
